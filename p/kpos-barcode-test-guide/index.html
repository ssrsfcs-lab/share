POS 설정 수정 테스트

20으로 시작하는 바코드를 찍으면 토탈세일즈 화면으로 넘어가는 문제를 고치기 위한 테스트입니다. 라벨은 그대로 두고, POS 화면에서 체크박스 1개만 끕니다.

⏱ 소요 약 10분 🕐 한가한 시간에 진행 ☑ 파일 수정 없음 — 화면 클릭만 ↩ 문제 시 체크 다시 켜면 원복
준비

상품 3개를 계산대에

ㄱ

바코드가 20으로 시작하는 상품 1개

예: SSOBE 텐셀 티셔츠 등 박스히어로 라벨 (2006066··· 처럼 20으로 시작)

ㄴ

바코드에 영문이 섞인 상품 1개

예: 쉬즈미스 · 리스트 · 모리모리스 (SSKPON··· 처럼 영문 포함)

ㄷ

평소에 잘 찍히던 아무 상품 1개

기존 판매가 그대로 정상인지 확인하는 용도

1단계

설정 바꾸기 — POS 1대씩

1

K_Pos에서 Basic Setup 화면 열기

관리자 메뉴의 기본설정(Basic Setup). 비밀번호를 물으면 평소 관리자 비밀번호를 입력하세요.

2

맨 오른쪽 Weight Setup 탭 → use Weight Code 체크 끄기 ★핵심

끄기 전에 원래 상태(켜짐/꺼짐)와 그 아래 칸의 숫자(예: 21, 21)를 사진 찍거나 메모해주세요.

3

Device Setup 탭(맨 왼쪽) → Use Character Barcode 확인만

체크돼 있으면 그대로 둡니다. 꺼져 있을 때만 체크하고, 원래 꺼져 있었다고 메모해주세요.

4

Save 누르고 K_Pos 재시작

화면 하단 안내대로 저장 후 프로그램을 껐다가 다시 실행합니다.

매장에 POS가 2~3대면 각 단말마다 1~4번 반복 — 한 대만 바꾸면 그 대만 고쳐집니다
2단계

확인 스캔 — 판매 화면에서 3개 찍기

순서스캔할 것정상이라면결과
ㄱ20으로 시작하는 상품SSOBE 등상품이 떠야 함예 / 아니오
ㄴ영문 섞인 상품쉬즈미스·리스트·모리상품이 떠야 함예 / 아니오
ㄷ평소 잘 찍히던 상품평소처럼 떠야 함예 / 아니오
✅ 셋 다 "예"라면ㄱ 상품으로 결제 1건을 끝까지 완료한 뒤 바로 취소(Void) 하고, 그 시간을 메모해주세요.
⚠ ㄱ이 여전히 토탈세일즈로 튀면거기서 멈추고 요청한 담당자에게 알려주세요. 되돌릴 필요는 없습니다.
🚨 ㄷ(기존 상품)이 안 찍히면Weight Setup 탭에서 use Weight Code를 다시 체크 → Save → 재시작하면 원래대로 돌아갑니다. 그리고 담당자에게 알려주세요.
